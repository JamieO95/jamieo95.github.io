I must start with, I'm a little disappointed. I went trawling through old files but it didn't come up as fruitful as I would have liked. I found this one website I made. Although it was fully explored within minutes, those minutes were painful. I've spent some time sanitizing it, I recall it being a business owner who came in to see if any students could build them a mobile-friendly website as they already had a website that was just fine for desktop usage. There's even a (dead) link there to 'Desktop Website' that literally just pushed to the business owner's website. Wild.

To give myself some credit, it does actually look okay if you shrink the window horizontally. The JS, while technically works, is in every page individually and would be a nightmare if changes needed to be made. Similarly, so is every other bit of code minus the CSS (some inline stuff). Looking at this, and sanitizing it, made me very thankful Jekyll can handle that stuff in a single location.

Anyway, here it is. It's (now) called Mr Vehicle Guy, and I made it in 2011. I was 16. Enjoy.
